心得体会：
之前对 js 代码执行一直不理解，通过今天的学习，理解了js代码执行一段可执行代码会如何工作。
因为 js 采用词法作用域，所以函数作用域在函数声明时就确定了。
函数创建、执行过程如下：
1. 函数被创建 保存父级(词法层面上的父级)作用域到内部属性 `[[scope]]`
2. 执行函数 创建执行上下文，然后将函数的执行上下文压入执行上下文栈中。
3. 接下来进行函数执行前的准备工作
    + 复制函数的 `[[scope]]` 属性创建作用域链。
    + 使用 arguments 创建活动对象，然后初始化活动对象 加入形参、函数声明、变量声明
    + 将活动对象压入函数作用域链顶端
4. 执行函数 随着函数执行 修改变量对象的属性值
5. 执行完毕 执行上下文栈弹出函数执行上下文

心得体会
1. 节流和防抖都是为了优化重复触发的点击事件。但是他们的原理不同，节流是在持续触发事件时，每隔 n 秒执行一次。防抖是在停止触发 n 秒后，执行事件。
